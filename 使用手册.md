# 趣学堂（FunLearn）使用手册

## 一、项目简介

趣学堂是一款面向幼小衔接阶段（3-7岁）儿童的互动学习应用，涵盖拼音、汉字、英语、数学和科普探索五大模块。采用游戏化学习路径设计，参考洪恩识字的学习流程，支持语音识别跟读、手写练习、笔画演示、间隔重复复习等功能。

**技术栈：** Vue 3 + TypeScript + Vite + Tailwind CSS + Pinia

---

## 二、环境要求

| 依赖 | 最低版本 |
|------|---------|
| Node.js | 18.0+ |
| npm | 9.0+ |
| 现代浏览器 | Chrome 80+ / Safari 14+ / Edge 80+ |

> **注意：** 语音识别和录音功能需要浏览器支持 Web Speech API 和 MediaRecorder API，推荐使用 Chrome 浏览器。这些功能需要在 HTTPS 或 localhost 环境下才能使用。

## 三、安装与启动

### 1. 安装依赖

```bash
cd studySite
npm install
```

### 2. 本地开发启动

```bash
npm run dev
```

启动后终端会显示：

```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.x.x:5173/
```

#!/bin/bash
WSL_IP=$(hostname -I | awk '{print $1}')
echo "WSL IP: $WSL_IP"
echo "请在 Windows 管理员 PowerShell 中执行："
echo "netsh interface portproxy delete v4tov4 listenport=5173 listenaddress=0.0.0.0"
echo "netsh interface portproxy add v4tov4 listenport=5173 listenaddress=0.0.0.0 connectport=5173 connectaddress=$WSL_IP"


- **本机访问：** 打开浏览器访问 `http://localhost:5173/`
- **局域网访问：** 手机/平板等设备访问 `http://192.168.x.x:5173/`（需与电脑在同一网络）

> 项目已配置 `host: '0.0.0.0'`，默认支持局域网访问。

### 3. 生产构建

```bash
npm run build
```

构建产物输出到 `dist/` 目录，可部署到任意静态服务器。

### 4. 预览生产构建

```bash
npm run preview
```

### 5. 类型检查

```bash
npm run check
```

## 四、功能模块详解

### 4.1 欢迎页 — 创建角色

首次打开应用会进入欢迎页：

1. **输入名字：** 填写小朋友的名字（最多8个字）
2. **选择头像：** 从 🐰🐱🐶🦊🐼🐸 六个可爱头像中选择
3. **点击"开始学习之旅"：** 创建角色并进入首页

如果已有多个角色，可以在欢迎页选择切换，也可以点击"添加新朋友"创建新角色。

---

### 4.2 首页 — 学习路径

首页展示五大学习模块，每个模块显示学习进度：

| 模块 | 图标 | 说明 |
|------|------|------|
| 拼音王国 | 📝 | 声母、韵母、整体认读 |
| 汉字花园 | 📖 | 幼小衔接常用汉字 |
| 英语世界 | 🌍 | 基础英语单词 |
| 数学城堡 | 🔢 | 数字认识和加减法 |
| 探索天地 | 🌟 | 自然科学和传统文化 |

首页还会显示：
- **复习提醒：** 有待复习的知识点时，会出现红色提醒卡片
- **今日挑战：** 点击可进入综合测验
- **星星数量、连续学习天数、等级**

---

### 4.3 拼音王国

**学习内容：**
- 声母（b, p, m, f, d, t, n, l, g, k, h, j, q, x, zh, ch, sh, r, z, c, s, y, w）
- 韵母（a, o, e, i, u, ü, ai, ei, ui, ao, ou, iu, ie, üe, er, an, en, in, un, ün, ang, eng, ing, ong）
- 整体认读（zhi, chi, shi, ri, zi, ci, si, yi, wu, yu, ye, yue, yuan, yin, yun, ying）

**学习流程（5步法）：**

1. **看图识音 👀** — 展示拼音和配图，点击"听一听发音"播放语音
2. **发音练习 🎤** — 点击麦克风按钮跟读，系统通过语音识别判断发音是否准确；录音后可点击播放按钮回听自己的发音
3. **书写练习 ✍️** — 在三线格中书写拼音，需写3次；可先点击"看笔顺"观看演示
4. **组词示例 💬** — 展示包含该拼音的词语，点击词语可听发音
5. **巩固挑战 🏆** — 选择题测试，从四个选项中选出正确的拼音

---

### 4.4 汉字花园

**学习内容：** 60个幼小衔接常用汉字，分为数字、人物身体、自然、生活等类别。

**学习流程（5步法）：**

1. **看图识字 👀** — 展示大字、拼音和配图，点击"听一听发音"播放语音
2. **字源故事 📜** — 讲解汉字的造字来源，点击"听故事"播放语音朗读
3. **书写练习 ✍️** — 在田字格中书写汉字，需写3次：
   - 上方有笔画演示动画，点击播放可一笔一划观看书写顺序
   - 下方田字格供手写练习
4. **组词造句 💬** — 展示组词和造句，每个都有语音播放按钮：
   - 点击组词按钮听词语发音
   - 点击"听句子"听整句朗读
   - 选择题：选出包含该字的词语
5. **巩固挑战 🏆** — 看图选字，从四个汉字中选出正确答案

**完成后：**
- 获得星星奖励
- 可选择"学习下一个"继续学习，或"返回首页"

---

### 4.5 英语世界

**学习内容：** 基础英语单词，包括水果、动物、颜色、数字、家庭等分类。

**学习流程（5步法）：**

1. **看图识词 👀** — 展示单词、音标、中文翻译和配图，点击播放英语发音
2. **跟读练习 🎤** — 点击麦克风跟读，系统通过语音识别判断发音：
   - 识别成功显示分数和评价
   - 录音可回放，点击播放按钮听自己的发音
3. **书写练习 ✍️** — 在三线格中书写英文单词，需写3次
4. **例句学习 💬** — 展示例句，点击播放英语朗读
5. **巩固挑战 🏆** — 选择题测试

## 五、核心功能说明

### 5.1 学习路径系统

- 拼音和汉字模块采用顺序解锁机制
- 必须完成当前内容才能解锁下一个
- 进度自动保存在本地

### 5.2 间隔重复复习

- 采用 SM-2 算法自动安排复习
- 根据答题质量调整复习间隔
- 首页会提醒需要复习的知识点数量
- 点击提醒可直接进入复习测验

### 5.3 语音功能

| 功能 | 说明 | 使用方式 |
|------|------|---------|
| 语音播放 | 朗读文字内容 | 点击 🔊 播放按钮 |
| 语音识别 | 判断跟读发音 | 点击 🎤 麦克风按钮后说话 |
| 录音回放 | 听自己的发音 | 跟读后点击 ▶ 播放按钮 |

> **注意：** 语音功能需要浏览器授权麦克风权限，且需要在 HTTPS 或 localhost 环境下使用。

### 5.4 书写练习

| 类型 | 格子 | 书写次数 | 笔画演示 |
|------|------|---------|---------|
| 拼音 | 三线格 | 3次 | ✅ |
| 汉字 | 田字格 | 3次 | ✅ |
| 英语 | 三线格 | 3次 | ✅ |

- **笔画演示：** 点击播放按钮，可观看一笔一划的书写动画
- **手写练习：** 用手指或鼠标在格子中书写
- **清除重写：** 点击清除按钮可重新书写

## 七、常见问题

### Q: 语音识别不工作？
A: 请确保：
1. 使用 Chrome 浏览器
2. 已授权麦克风权限
3. 通过 HTTPS 或 localhost 访问
4. 设备有可用的麦克风

### Q: 局域网无法访问？
A: 请确保：
1. 手机/平板与电脑在同一 WiFi 网络
2. 电脑防火墙允许 5173 端口访问
3. 使用终端显示的 Network 地址访问

### Q: 图片加载失败？
A: 部分图片依赖外部 API 生成，需要网络连接。如加载缓慢请耐心等待。

### Q: 如何重置学习数据？
A: 在浏览器开发者工具中清除 localStorage，或清除浏览器缓存即可。

### Q: 忘记家长密码？
A: 默认密码为 `123456`。如已修改，可在浏览器开发者工具的 Application → Local Storage 中删除 `funlearn_parent_password` 键来重置。
